About Page

This is a description of what this site is designed to do.

This site was created using a flask server as well as various packages that flask offers.

This site is designed to allow a user to create an account, login to their account, and access/manage their own personalised RSS feed.

This website uses https://rss.bloople.net/ to convert the RSS feed links into html code to be shown on the website. On the homepage it advertises this ability with a lack of other restrictions.

WTForms was used to increase security on the login page as it provides a Secure Form with CSRF token, as well as Global CSRF protection.
